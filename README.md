# Autocomplete in ASP.NET Core — Native datalist and fetch

Companion sample for the GeeksArray tutorial
[Autocomplete in ASP.NET Core — Native datalist and fetch](https://geeksarray.com/blog/jquery-ajax-autocomplete-in-asp-net-mvc-core).

Debounced autocomplete against a JsonResult suggestions endpoint using the browser-native datalist element — run with `dotnet run` and open /Catalog. No jQuery UI required.
